# Elementos estéticos del bonsái
Los artículos [La descuidada elegancia de la naturaleza. Ideal estético japonés](https://aprendiendobonsai.com/la-descuidada-elegancia-de-la-naturaleza-ideal-estetico-japones/) y [Elementos estéticos del bonsái](https://aprendiendobonsai.com/elementos-esteticos-del-bonsai/) introducen a la estética que conforma la apariencia y percepción de nuestros bonsáis. Aplicar los ideales y elementos estéticos de manera efectiva captura la atención del espectador y transmite la intención de forma impactante.
Al crear un bonsái, este debe ser una evocación a través de la cual expresemos qué significa la naturaleza para nosotros. Para ello, debemos extraer los elementos de la naturaleza que emocionan.

La tabla [Elementos estéticos del bonsái](https://github.com/AprendizBonsai/Elementos-esteticos-del-bonsai/blob/main/Elementos%20esteticos%20del%20bonsai.jpg) que hay en este repositorio es una versión de alta resolución que se utiliza en el segundo artículo.
